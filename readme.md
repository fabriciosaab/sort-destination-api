# Sort Destination API
## Description

API to Sort Tickets destinations to human readable format

## Project setup

```bash
$ gh repo clone fabriciosaab/sort-destination-api
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test
```
